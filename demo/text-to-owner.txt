HOW TO TURN THE FILE INTO A LINK YOU CAN TEXT
=============================================

You've got three easy options. Pick whichever takes you less than 5 minutes.

Option A — GitHub Pages (free, permanent link, recommended)
-----------------------------------------------------------
The repo is already on GitHub, so this is mostly clicks.

  1. Commit and push the demo folder:
       git add demo/owner-demo.html
       git commit -m "Add owner demo preview"
       git push

  2. On github.com, open the repo → Settings → Pages.
  3. Source: "Deploy from a branch". Branch: main, folder: / (root). Save.
  4. Wait ~1 minute. GitHub gives you a URL like:
       https://<your-username>.github.io/city-bucket-list/demo/owner-demo.html
  5. Paste that link into the SMS above and send.

Option B — Drop into OneDrive and share a link (also free)
----------------------------------------------------------
  1. The file already lives in your OneDrive folder.
  2. Right-click owner-demo.html in File Explorer → Share → "Copy link".
  3. Make sure permission is "Anyone with the link can view".
  4. Paste that link into the SMS.

  Note: some phones open OneDrive files in the OneDrive app instead of a
  browser, which can break the layout. If the owner mentions weirdness,
  switch to Option A.

Option C — Quick tunnel for a live demo (laptop must stay open)
---------------------------------------------------------------
Good if you want him to see it RIGHT NOW while you're together.

  1. Open a terminal in the project folder and run:
       npx http-server ./demo -p 8080
  2. In a second terminal:
       ngrok http 8080
  3. ngrok prints a public URL — paste that + "/owner-demo.html" into the SMS.
  4. Stop when done (link dies when you stop ngrok).
